## Scripts Included

| Script | Description |
|--------|-------------|
| `brew_tap_audit.py` | Audit Homebrew taps for accessibility and installed packages |
| `create_letter_logo.py` | Create letter-based logos using Pillow |
| `gkeepapi_script.py` | Google Keep API integration example |
| `gpsoauth_script.py` | Google Play Services OAuth authentication |
| `gpsoauth_script_alternative.py` | Alternative OAuth token exchange method |
| `pip_cache_directories.py` | Display pip cache directory locations |
| `infinteRun.py` | Simple infinite loop script |

## Technologies Used

- **Python 3** - Main programming language
- **Pillow** - Image processing library
- **gkeepapi** - Google Keep API client
- **gpsoauth** - Google Play Services OAuth

## Infrastructure

- **Docker** - Container support
- **Vagrant** - Development environment
- **Azure Pipelines** - CI/CD

## Installation

```bash
pip install -r requirements.txt
```
